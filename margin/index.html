3. Margin Percetage

Margin percetage is relative to its container

margin-top: 10%;

margin-left: 10%;

4. Margin Collapse

Margin collapse only happens with block element and in vertival direction.

Ways to avoid margin collapse: set inline-block, add border, add padding to the container
